Understanding Reports

SQL compliance manager Reports allows you to glean various information and trends from your audit data, such as user activity history and privilege changes on SQL Server objects. This user interface leverages Microsoft Reporting Services, providing secure and easy access to your audit data. Use SQL compliance manager Reports to track compliance on demand and provide self-service reporting to third-party auditors. For more information, see Report on Audit Data.
