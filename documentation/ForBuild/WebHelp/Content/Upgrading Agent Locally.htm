Upgrade the SQLcompliance Agent locally

You can use the SQL Compliance Manager setup program to upgrade the SQLcompliance Agent on the local computer that is running the registered SQL Server instance. Use this approach when you are upgrading the SQLcompliance Agent on a registered SQL Server where you manually installed the agent. For more information, see Upgrading SQL Compliance Manager.
