Understanding the Management Console

The Management Console allows you to quickly and easily perform any of the following tasks:

  • View the activity summary for your enterprise, SQL Server instance, and individual database
  • Configure SQL compliance manager for your specific needs
  • Secure audit data using SQL Server logins
  • Administer audit settings to ensure continual compliance
  • Manage archived audit data

The Management Console layout consists of the following panes:

Navigation tree

Allows you to quickly navigate to specific features and tabs using a standard tree control.

View pane

Displays data dynamically, depending on the console tree node you selected.
